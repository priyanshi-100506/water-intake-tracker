## 🚀 Quick Start (5 Minutes)

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Start Three Services

**Terminal 1 - Ollama:**
```bash
ollama serve
```

**Terminal 2 - API Server:**
```bash
python -m uvicorn src.api:app --host 127.0.0.1 --port 8000 --reload
```

**Terminal 3 - Dashboard:**
```bash
streamlit run dashboard_new.py
```

### 3. Open Dashboard
```
http://localhost:8501
```

### 4. Create User & Start Tracking
- Click "New User"
- Fill in your profile
- Log water intake
- View analytics!

## 🔌 API Endpoints

```
Health & Docs:
  GET  /                    - Health check
  GET  /docs                - API documentation

User Management:
  POST   /profile           - Create user
  GET    /profile/{id}      - Get profile
  PUT    /profile/{id}      - Update profile
  DELETE /user/{id}         - Delete user

Goals:
  POST   /goal              - Set hydration goal

Intake:
  POST   /log-intake        - Log intake (+ AI analysis)
  GET    /history/{id}      - Get intake history

Analytics:
  GET    /analytics/{id}    - Get metrics
  GET    /weekly-report/{id} - Get AI report
```
